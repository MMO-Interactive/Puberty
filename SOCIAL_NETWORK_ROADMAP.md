# Social Network Transition Roadmap

This project started as an education + private tools site and is now evolving into a social learning network.

## Completed foundation

- Community feed with posts, likes, and comments.
- User profiles with follow/unfollow.
- Notifications for follows, likes, and comments.
- Dedicated content hubs (Resources + Sex Education module).

## Next milestones

1. **Account identity**
   - Profile photos and optional pronouns.
   - Public vs private profile toggles.

2. **Social graph**
   - Mutual friendship mode for younger users.
   - Follow-request approval flow.

3. **Community safety**
   - Word filters and report-abuse workflow.
   - Moderator queue and action history.

4. **Content discovery**
   - Hashtags and topic channels.
   - Search for posts, people, and modules.

5. **Messaging**
   - Safe direct messaging with guardian controls.
   - Block/mute tools and message request inbox.

6. **Engagement**
   - Daily challenges tied to education modules.
   - Achievement cards shareable to the community feed.

7. **Observability**
   - Basic analytics: retention, lesson completion, safety reports.
   - Error reporting and audit logs for moderation actions.
